HTML Quick Guide

Above is called a header.
<h1> is used for the biggest header
<h6> is used for the biggest header

This is the body. Example code for an html page is this:
< !DOCTYPE html >
  < html >
    < head >
    < title >
     Put the title here
    < /title >
    < /head >
   < body >
     Put the body here
   < /body >
  < /html >

Use '< br >' to break line, and must be typed after every line break.

&nbsp; is a regular space.

&ensp; is a two spaces gap.

&emsp; is a four spaces gap.

<br> will make a new space for whatever you want to write.
<p> and </p> is a paragraph and will run on horizontally as long as you have space.
<pre> and </pre> will use the new lines and spaces that are used inside the text.

Even though HTML and CSS are different languages, they are used together. Code for CSS is written inside curly brackets { CSS }.
